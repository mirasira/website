Tmux Commands

Start a new session

tmux

Attach to an existing session

tmux a

Default prefix key

$prefix (ctrl+b)

Detach from the session while it keeps running

$prefix + d

Create new window

$prefix + c

List windows with option to select one to see

$prefix + w

Next window

$prefix + n

Previous window

$prefix + p

Name window

$prefix + ,

Split vertically

$prefix + %

Split horizontally

$prefix + "

Scrolling

$prefix + [

Kill window

$prefix + &
