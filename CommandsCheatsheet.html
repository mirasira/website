Job Management

My jobs

squeue -u $USER 
squeue --me

Get job info with time limit

squeue --me -l

Cancel job

scancel <job_id>

Submit batch with job

sbatch script.sh

Modify job's time limit

scontrol update JobId=JOBID timelimit=4:00:00

Allocate Interactive Job

Run interactive job with cpu, one hour:

salloc -A OPEN-29-7 -p qcpu_exp

Run interactive job with gpu, one hour:

salloc -A OPEN-29-7 -p qgpu_exp

For 24 hours

salloc -A OPEN-29-7 -p qgpu

For more GPUs

salloc -A OPEN-29-7 -p qgpu --gpus 8

Set time

salloc -A OPEN-29-7 -p qgpu --gpus 8 --time 24:00:00
gpus NUM_OF_GPUS TIME
cpus TIME

final url format

Download folder from Google Drive

final url format

gdown --folder https://drive.google.com/drive/folders/10mDNZnYrg55ZiP9GV4upKWnxlxay1OwM

C

Compile C code

distance.c is name of the file, distance is name of the executable

Compile C code

gcc distance.c -o distance -lm

Compile C code with debugging

gcc -g distance.c -o distance -lm

Compile C code with debugging and warnings

gcc -g -Wall distance.c -o distance -lm
